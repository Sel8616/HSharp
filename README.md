#SHC

###⿻ About
***
__SHC__ is a simple and lightweight HTTP client based on java.net.HttpURLConnection.

[--> Maven <--]


###♨ Features
***
* Singleton http client.

* Multi-process for concurrent requests.

* Request can be false canceled.


###☀ Notice
***
* JDK1.8+ is Mandatory.
